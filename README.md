# Design of a Sensor based Posture Classifier (Posture Track)
## Abstract
Low back pain is a problem often associated with poor posture and prolonged sitting. With
increased use of sedentary work environments, there is a growing need for effective posture
monitoring systems. This project presents the design and development of a posture monitoring
system using a capacitive touch sensor and a time-of-flight distance sensor integrated with a
microcontroller. 
The system detects user posture based on proximity or near contact and
distance measurements and provides real-time feedback through visual and auditory alerts
when incorrect posture is maintained for a specified duration. Data is transmitted to a cloud
platform for monitoring and analysis, enabling the evaluation of posture behavior over time.

The system is tested under different posture settings and the results demonstrate its ability to
distinguish between correct and incorrect posture effectively.
The system is designed to be non-obstructive(non contact) ,gaining from disadvantages
presented by other alternatives in the market while giving real time actionable insights to the
user.
